# FontBlitz ⚡

A lightning-fast browser extension that lets you change fonts on any website instantly.

## Features

- Instant font switching
- Change fonts on any website in real-time
- Access to hundreds of Google Fonts
- Works on all websites
- Simple and intuitive interface

## Installation

### From Chrome Web Store
1. Visit the [Chrome Web Extension Store](https://chromewebstore.google.com/detail/fontblitz-font-changer/ndhikidjijghfocidpnidbgkhioiajde)
2. Click "Add to Chrome"
3. Pin the extension for easy access


## Development

```bash
# Install dependencies
pnpm install

# Run development mode with hot reload
pnpm dev

# Build for production
pnpm build
```


## Tech Stack

- **React**
- **Vite**
- **Javascript**
- **Chrome Extension API**


## Permissions

FontBlitz requires the following permissions:

- `tabs` and `scripting` - To modify fonts on the current page
- `storage` - To save theme preference
- `fontSettings` - To get system fonts
